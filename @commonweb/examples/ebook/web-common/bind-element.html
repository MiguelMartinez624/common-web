Bind Element 
<bind-element>
Descripcion
Es un CustomElement que funciona como enlace entre dos components, puede ser para pasar informacion de un evento a otro componente o para encadenar eventos entre componentes StaticTemplate

Uso

Este codigo creara una relacion entre el evento "click" del <button> al metodo toggleClass del <div> y se pasara el valor "hidden" al metodo.



Atributos
Nombre Descripcion
from Selector de quien inicia el acoplamiento
to Selector del elemento objetivo.
value Valor estatico a pasar en caso de llamar un metodo en el elemento objetivo
input-src Selector a un elemento de donde se tomara el valor para ser pasado al objetivo
input-path Ruta a la propiedad que se desea extraer antes de ser pasado al elemento afectado
