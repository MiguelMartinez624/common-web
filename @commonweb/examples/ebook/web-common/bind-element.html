Bind Element 
 <bind-element>

Descripcion
Es un CustomElement que funciona como enlace entre dos components, puede ser para pasar informacion de un evento a otro componente o para encadenar eventos entre componentes StaticTemplate

Uso

Atributos
Nombre Descripcion
from Contexto de datos que seran usados en el template y propagados en la interpolacion
to Ruta hacia el archivo/template (html) que sera insertando en el DOM
value Ruta hacia el archivo/template (html) que sera insertando en el DOM
input-src Ruta hacia el archivo/template (html) que sera insertando en el DOM
input-path Ruta hacia el archivo/template (html) que sera insertando en el DOM
