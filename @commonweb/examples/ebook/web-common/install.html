Installation

Es un framework construido sobre WebComponents sin ninguna dependencia externa que facilita y simplifica la creacion de web app abstrayendo tareas comunes como llamadas a api, renderizar en ciclos, enlace de eventos y mas, mediante el uso de WebComponents que pueden ser usados como bloques principales par la construccion de tus propios componentes.
