Query Expression

Descripcion

Es un patron de busqueda, que indica la ruta a un elemento y a una propiedad/metodo/evento de ese elemento. Cuenta de algunas palabras claves o identificadores reservados que se refieren a elementos especificos en el arbol de elementos



Identificadores Reservados
Nombre Descripcion
@host Se refiere al FrameworkComponent mas cercano de forma ascendente se considera este el host del elemento, y el contexto esta atado a este
Ej @host:[data.imageURL]
  • static-template
    • div
      • div
        • h4
      • div
        • img src="@host:[data.imageURL]"
En este caso podemos ver que desde el elemento img referimos al host que en este caso seria el static-template y de este tomamos la propiedad de data.imageURL
