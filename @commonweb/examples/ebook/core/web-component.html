Web Component

Descripcion

Es un custom element con algunos metodos y funcionalidades predefinidas como lo son interpolacion , callbacks para atributos de forma sencilla e integracion con directivas.



Declaracion

Para declarar un WebComponent lo unico que tenemos que hacer es extender una clase ya se bien con la clase nativa HTMLElement o la clase expuesta por el framework FrameworkComponent 

            @WebComponent({
                selector:'hello-word-component'
                template:`<h5>Hello World! </h5>`
            })
            export class HelloWorldComponent extends HTMLElement{}


Esta es la declaracion basica de un componente se registrara automaticamente dentro de los elementos del navegador, por lo que sigue las mismas reglas.
